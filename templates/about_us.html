What is See U Monday?

See U Monday (SUM) is a personalized job searching tool where the results are tailored to individual users based on their resume. Using SUM, the user can upload a resume in pdf format. The data available from that input resume will be parsed through the text to identify user’s skill set, and match the skill set with our documents that we have already built by crawling current top job searching tools such as Glassdoor and Indeed.

For more information, please click: Read Me

Data Acquisition

We crawl the top 3 job sites: Indeed, Monster and Glassdoor. We create a list of potential job titles and iterative through the list by changing query parameters on the urls to crawl. Each job posting is a document and we store the job title, company name, location, and job description.
